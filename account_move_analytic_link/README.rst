This module allows users to navigate from posted journal items that have
analytic distribution assigned to the analytic items generated.

**Table of contents**

.. contents::
   :local:

Usage
=====

- In a posted move, go to the move line with analytic distribution
  assigned and click on the Open analytic items button to navigate to
  the related Analytic Lines. The same button could be found in the
  journal items view.
