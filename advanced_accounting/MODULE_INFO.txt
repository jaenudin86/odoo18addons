DESCRIPTION
================================================================================

Advanced Accounting module brings enterprise-level accounting features to 
Odoo 18. This comprehensive module provides powerful tools for budget 
management, asset tracking, cost center analysis, and payment follow-up.

================================================================================
KEY FEATURES
================================================================================

✓ Budget Management
  - Multi-level budget creation and approval
  - Budget vs Actual tracking in real-time
  - Budget alerts and warnings
  - Comprehensive budget reports

✓ Asset Management
  - Fixed asset tracking and lifecycle management
  - Automated depreciation (Linear & Degressive methods)
  - Depreciation board with automatic journal entries
  - Prorata temporis calculation
  - Asset disposal functionality

✓ Cost Center Management
  - Hierarchical cost center structure
  - Department-wise cost tracking
  - Budget allocation per cost center
  - Performance analysis and reporting

✓ Payment Follow-up
  - Automated payment reminder system
  - Multi-level follow-up configuration
  - Customizable email templates
  - Follow-up history tracking

✓ Enhanced Analytic Accounting
  - Extended analytic account features
  - Budget integration with analytics
  - Cost center linkage
  - Advanced reporting capabilities

================================================================================
INSTALLATION STEPS
================================================================================

1. Download and extract module to Odoo addons directory
2. Update Apps List in Odoo
3. Search for "Advanced Accounting"
4. Click Install button
5. Configure settings and permissions

For detailed installation instructions, see INSTALL.md

================================================================================
QUICK START
================================================================================

After installation:

1. Create Your First Budget:
   Advanced Accounting > Budgets > Create

2. Add Your First Asset:
   Advanced Accounting > Assets > Create

3. Setup Cost Centers:
   Advanced Accounting > Cost Centers > Create

4. Configure Follow-up:
   Advanced Accounting > Payment Follow-up > Follow-up Levels

For detailed usage instructions, see USER_GUIDE.md and QUICKSTART.md
